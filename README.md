# AstrBot Hextech Plugin (海克斯乱斗信息差)

这是一个用于查询《英雄联盟》极地大乱斗（ARAM）中英雄海克斯联动数据的 AstrBot 插件。它通过爬取 [Apexlol](https://apexlol.info/) 的数据，为用户提供英雄的海克斯符文推荐、评级及分析。

## ✨ 主要功能

*   **智能英雄识别**：支持输入英雄别名、外号、数字代码（如“4396”、“大腰子”、“快乐风男”），插件会自动使用 LLM 识别为标准英雄名。
*   **海克斯联动查询**：获取英雄专属的海克斯联动数据，包括符文名称、阶级（棱彩/黄金/白银）及评分。
*   **评级标注**：
    *   🔥 **S 级 (强力!)**：高亮显示强力推荐的海克斯。
    *   🚫 **D 级 (陷阱!)**：警示不推荐选择的海克斯。
    *   ✨ **A 级** / ⚖️ **B 级**：清晰的层级展示。
*   **详细分析**：提供每个海克斯的具体联动说明和玩法建议。

## 🚀 安装

1.  将插件放置在 AstrBot 的插件目录下。
2.  安装所需的 Python 依赖：
    ```bash
    pip install aiohttp beautifulsoup4
    ```
3.  重启 AstrBot。

## ⚙️ 配置

插件支持通过 `enable_llm_search` 配置项开启或关闭 LLM 智能识别功能（默认开启）。如果未配置 LLM Provider，将仅支持标准英雄名查询。

## 💬 使用方法

在群聊或私聊中发送指令即可：

### 查询英雄信息

**指令格式**：
```
/海斗 <英雄名或别名>
```

**示例**：
*   **标准名查询**：
    > 用户：/海斗 诺手
    > 
    > Bot：🔍 正在查询【诺克萨斯之手 德莱厄斯】的相关信息...
    > 
    > 英雄: 诺克萨斯之手 德莱厄斯 (Darius)
    > ID: Darius
    > 
    > ⚡ **海克斯联动分析**
    > 🔥 【S 级】 **利刃华尔兹** (棱彩阶) **(强力!)**
    >    └ 💡 施放期间处于无敌状态，并在结束时满层流血...

*   **别名/黑话查询**：
    > 用户：/海斗 4396
    > 
    > Bot：🔍 正在查询【盲僧 李青】的相关信息...
    > ...
